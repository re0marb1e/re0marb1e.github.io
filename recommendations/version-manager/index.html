多版本环境管理

你可能会遇到一些情况需要在同一个操作系统上安装不同版本的语言环境，比如你想体验一下最新的Java10，同时某些软件只支持Java8，于是你必须同时保留两个环境，同时你还需要能够在操作系统上切换两个版本的Java环境。对于上述问题，很多语言都有相对应的工具来帮助你轻松解决，比如jEnv，rvm等。下面整理了各种语言的多版本环境管理工具的介绍以及简单的使用说明。

    • 通过jenv remove <jdk-version>移除指定的Java环境
    • 通过jenv global <jdk-version>配置全局Java环境
    • 通过jenv local <jdk-version>配置本地Java环境
    • 通过jenv shell <jdk-version>配置ShellJava环境
