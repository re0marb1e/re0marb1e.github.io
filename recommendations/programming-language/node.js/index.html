Node.js阅读推荐

书籍推荐

  • 《Node.js in Action, Second Edition》
    一本不错的Node.js入门书，但本书有部分内容已过时，比如关于koa，kraken，hapi的部分等。

  • 《An Introduction to libuv》

  • 《libuv中文教程》

  • 《深入理解Node.js：核心思想与源码分析》

  • 《Node.js 调试指南》

  • 《Express in Action》
    讲述Express框架的书，想使用Node.js做Web开发都可以看一看。

  • 《Sails.js in Action》
    讲述Sails.js框架的书，如果对Sails.js有兴趣可以看看。
