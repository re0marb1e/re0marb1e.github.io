属性特性

ECMAScript定义了一些属性特性用于规范JavaScript引擎的内部实现，这些属性特性不能够通过JavaScript直接访问到。不过JavaScript提供了Object.getOwnPropertyDescriptors(obj)和Object.getOwnPropertyDescriptor(obj, prop)两个方法，通过这两个方法可以获取到对应的对象属性描述符，以此可以访问到内部的属性特性。

此外，JavaScript还提供了Object.defineProperties(obj, props)和Object.defineProperty(obj, prop, descriptor)两个方法，通过设置属性的属性描述符对象，以此来设置对象的属性特性。

ECMAScript一共定义了六个对象属性特性，他们分别是：

属性特性 描述 默认值
Configurable 表示能否删除属性，改变属性的类型以及修改属性的特性 false
Enumerable 表示能否通过for…in循环返回属性 false
Writable 表示能否修改属性的值 false
Value 属性的值，可以是任何JavaScript类型 undefined
Get 在读取属性时调用的函数 undefined
Set 在写入属性时调用的函数 undefined

数据属性与访问器属性

根据拥有的属性特性的不同，对象属性又分为数据属性与访问器属性。其中数据属性拥有[[Configurable]], [[Enumerable]], [[Writable]]和[[Value]]四个特性；访问器属性拥有[[Configurable]], [[Enumerable]], [[Get]]和[[Set]]四个特性。

数据属性不能拥有访问器属性的[[Get]]和[[Set]]两个特性，且访问器属性不能拥有数据属性的[[Writable]]和[[Value]]两个特性。
